# Proyecto TS — Dashboard mínimo

## Requisitos
- Node >= 18 (recomendado 22.12+)
- TypeScript como devDependency

## Pasos
```bash
npm init -y
npm install typescript --save-dev
npx tsc --init   # puedes sustituir por el tsconfig incluido
npx tsc
# sirve la carpeta actual; por ejemplo:
npx http-server . -o
```
Abre `index.html` (carga `dist/main.js` compilado).
