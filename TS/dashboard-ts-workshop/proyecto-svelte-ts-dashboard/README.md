# Svelte + TypeScript — Dashboard mínimo (Vite, no SvelteKit)

## Requisitos
- Node **>= 22.12** (para `@sveltejs/vite-plugin-svelte@6.2.x` y Vite 6)
- npm 10+

## Pasos
```bash
npm install
npm run dev
```
- Tailwind v4 está habilitado vía `@tailwindcss/vite` y `@import "tailwindcss";` en `src/styles.css`.
